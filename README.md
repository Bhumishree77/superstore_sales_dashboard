# Superstore Sales Dashboard

## Project Overview
This project is an interactive Power BI dashboard built using the Superstore dataset. It provides insights into sales, profit, orders, and regional performance.

## Dashboard Features
- Total Sales KPI
- Total Profit KPI
- Total Orders KPI
- Sales by Category
- Sales by Region
- Monthly Sales Trend
- Profit by State
- Interactive Slicers (Region, Category, Segment)

## Tools Used
- Microsoft Power BI
- Superstore Dataset
